Web browser servers & client

Client:Technically the client simply means the users device e.g computer, smart phone, Tv and its also the sofware you use to access the web.This includes the web browser.When you type a url into your browser and hit enter.You are acting as the client sending a request

Web-browser:Web browser is the application software on the client device e.g Google Chrome Firefox, opera, saferi that act as an interface to the web.Its job is to

1. Send a request to a web server for a specific web-page or resource.

2.Recieve the server's response(Html, css,images e.t.c)

3.Render & display the content on your screen in a human readable format.

Web Server

Web Server: It is a computer or pieces of software that stores website & their content. when a client web browser sends a request,The web server processes it & send back the requested file .It is always connected to the internet and it is designed to handle multiple request from different client simulteneously .This entire process is called client-server model.

Ip addresses

IP address is a unique numerical label assigned to every device connected to a computer on the internet protocol for communication.It is like a street address for a computer on the internet .It allows computer to find & communicate with each other.A typical IP v 4 address e.g 192.168.1.1

Domain Name

A domain name is a human readable name that corresponds to one or more ip address For example Google.com is a domain name.Domain names have a hierachical structure for e.g .com is the top level domain and Google is a second level domain .

URL

A url is a complete web address that specifies the location of a resource on the internet and the protocol used to access it .
