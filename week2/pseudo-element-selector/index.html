Pseudo element

A css pseudo element is used to style specific part of an element, it is used to style the first letter or line of an element, insert content before or after an element, style the markers of list item, style the view box behind the dialog box

Pseudo element syntax

selector::pseudo-element{ property:value; }

The first line pseudo element

The first line pseudo element is used to add a special style to the first line of a text

First Letter Pseudo class

The ::first-letter pseudo-element is used to add a special style to the first letter of a text

Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolores, beatae!

The following properties apply to ::first-letter pseudo-element

  1. font property
  2. color property
  3. background property
  4. margin property
  5. padding property
  6. border property
  7. text decoration
  8. vertical align
  9. text transform
  10. line height
  11. float
  12. clear

The ::selection pseudo-element

The ::selection pseudo-element matches the portion of an element that is selected by a user

css property that can be applied to the ::selection pseudo-element are color, background, cursor, outline
