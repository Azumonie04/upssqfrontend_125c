The CSS flex-container

The css properties we use in flex-containers are:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

The Css flex-direction property

The flex-direction property specifies the display direction of flex items in a flex-container, the flex-direction property can have one of the following

  • row[that is default]
  • column
  • row reverse
  • column reverse

CSS flex-wrap property

The flex-wrap property specifies whether the flex item should wrap or not, if there's not enough room for them on one flex line. The flex wrap property can have one of the following values:

  • nowrap
  • wrap
  • wrap-reverse

The flex-flow property

The flex-flow property is a shorthand property for setting both the flex-direction and flex-wrap property

The css justify-content property

The justify-content property is used to align the flex-items when they don't use all available space on the main axis[horizontally]. The justify-content property can have one of the following values:

  • center
  • flex-start
  • flex-end
  • space-around
  • space-between
  • space-evenly

The CSS align-item property

The align-items property is used to align the flex item when they do not use all available spaces on the cross axis[Vertically]. The align-item property can have one of the following values:

  • center
  • flex-start
  • flex-end
  • strectch
  • baseline
  • normal

The CSS align-content property

The align-content property is used to align the flex lines. They align-content property is similar to align items, but instead of aligning flex-items, it align the flex lines. The align-content property can have one of the following values:

  • center
  • stretch
  • flex-start
  • space-around
  • space-between
  • flex-end
  • space-evenly
