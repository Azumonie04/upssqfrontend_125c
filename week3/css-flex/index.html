What is CSS-Flexbox

Flexbox is short for the flexible box layout module, flexbox is a layout method for arranging items in rows or columns, flexbox makes it easier to design a flexible responsive layout structures without using float or positioning

Flexbox vs Grid

The CSS-Flexbox layout should be used for one-dimensional layout, with rows or columns. The css GRID layout should be used for 2-dimensional layout, with rows and columns

CSS flexible box layout module

Before the flexbox layout module, they were 4 layout modes:

  • block, for sections in a web page
  • inline, for text
  • Table, for 2-dimensional table data
  • position, for explicit position of an element
